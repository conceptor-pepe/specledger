# specledger:new-change

Create a new SpecLedger change using repository templates.

Create:

- `changes/<date>-<change-name>/change.md`
- `changes/<date>-<change-name>/spec-delta.md`
- `changes/<date>-<change-name>/design.md`
- `changes/<date>-<change-name>/tasks.md`
- `changes/<date>-<change-name>/review.md`
- `changes/<date>-<change-name>/audit.md`
- `changes/<date>-<change-name>/test-review.md`
- `changes/<date>-<change-name>/commit-summary.md`
- `changes/<date>-<change-name>/archive.md`
