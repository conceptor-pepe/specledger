# Repository Description

## Short Description

Gate-driven engineering for AI-assisted software development.

## Long Description

Xiezhi is a repository-native engineering framework that turns AI-assisted development into a structured workflow with change artifacts, long-term specs, verification gates, delivery summaries, and archive steps.
